# Auto-Show FATE Helper

When entering Occult Crescent, automatically sends `/fatehelper` once - after the zone has finished loading - to open the FATE Helper window if it isn't already displayed. If the window is already open, nothing happens (no toggle).

Requires the [FATE Helper](https://github.com/teechep/FATEhelper) plugin by Teechep Bird. If FATE Helper isn't installed, this feature silently does nothing.
