# Occult Crescent Coffer Path Overlay

When the area map is open in South Horn, overlays the two community minimap-chest farming paths on top of it - red NW loop (26 chests) and blue E/S loop (42 chests). Bronze chests render as copper dots, silver chests as lighter dots. Chest data is taken from EurekaTrackerAutoPopper's `OccultChests.cs` (territory 1252) and the visit order matches the consolegameswiki Coffer Path image. Each path can be toggled independently, with line thickness, dot size, and opacity configurable.
