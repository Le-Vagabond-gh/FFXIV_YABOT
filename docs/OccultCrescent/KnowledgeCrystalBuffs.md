# Knowledge Crystal Buffs

Shows an "Apply Crystal Buffs" button next to Knowledge Crystals in Occult Crescent that swaps to each phantom job, casts the corresponding crystal buff (Pray, Counterstance, Romeo's Ballad, Quickstep), and switches back. Uses Inquiring Mind (Freelancer) when available to apply all 4 in one shot. Optional overlay buttons for Occult Treasuresight (auto-switches to Phantom Freelancer, casts, returns and remounts) and Geomancer Suspend (same flow, no remount). Also exposed as `/ykcb` / `/ycrystalbuffs`.
