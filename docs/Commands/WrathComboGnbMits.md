# WrathCombo: GNB Simple Mitigations Toggle

Slash command (`/ymits [on|off|toggle]`) and config-pane button that flips WrathCombo's GNB simple-rotation mitigations (Heart of Stone/Corundum, Aurora, etc.) in or out of the simple rotation - without modifying WrathCombo. Reflects directly into WrathCombo's live config dictionary so changes apply immediately and persist.
