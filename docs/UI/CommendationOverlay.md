# Commendation Tracker

After you leave a duty, shows your live commendation count (and `+N` for ones gained from that run) so you can see whether other players actually commended you. Commendations aren't delivered while the end-of-duty commend popup is up (so players can't pressure each other for them) - they only arrive once everyone has left the duty, which is why the overlay only appears after exit. Includes a brief golden glow + sparkle animation each time a commendation lands. Debug command: `/ycommend [reset]`.
