# Weather Forecast (WeatherWane)

FFXIV weather forecast panel - pick zones to track and see upcoming weather at a glance, with icon tooltips for weather names. Shows the current weather period highlighted, plus N future Eorzean weather periods (default 4, slider goes up to 16).

Adds a cloud icon (☁) to the server info bar when enabled:
- **Click** - toggle the forecast window
- **Shift-click** - open the zone-settings window

Originally a standalone plugin (`ffxiv_weatherwane`); folded into YABOT.
