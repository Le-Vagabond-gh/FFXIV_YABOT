# Auto-Sync FATEs

Automatically runs `/lsync` when you enter a FATE you're overlevelled for. Configurable per expansion (ARR/HW/SB/ShB/EW), can skip while in combat, and can also auto-confirm the FATE NPC start dialog. Defers the sync until any active NPC dialogue finishes so it doesn't interrupt cutscenes.

Can also auto-advance any NPC talk box while you're standing inside a FATE area (proximity to any active FATE in the zone), so escort/boss FATE dialogue gets clicked through automatically.
