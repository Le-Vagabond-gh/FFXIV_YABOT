# Login Improvements

A bundle of login / character-select quality-of-life features. Includes per-job emote loops on character select, auto-sheathe via the Sheathe Weapon emote, an in-screen emote name overlay, automatic battle stance on login, summoning your last pet/minion, skipping the launcher confirm and queueing the position dialog, and more. Each sub-feature is individually toggleable in the configuration pane.
