# WrathCombo: Tank Simple/Advanced Toggle

Slash command (`/yrot [simple|advanced|toggle]`) and config-pane buttons that switch every tank (PLD/WAR/DRK/GNB) between WrathCombo's Simple and Advanced rotations at once, for both single target and AoE. Enabling one mode automatically disables the other (they conflict in WrathCombo). Changes apply immediately and persist.

Advanced mode still uses whatever sub-options you've set up in WrathCombo - this only flips which top-level rotation each tank runs.
