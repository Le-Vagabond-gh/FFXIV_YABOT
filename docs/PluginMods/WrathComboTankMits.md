# WrathCombo: Tank Mitigations Toggle

Slash command (`/ymits [on|off|toggle]`) and config-pane button that flips WrathCombo's tank mitigations for all four tanks (PLD/WAR/DRK/GNB) at once. For each job it toggles both the Simple rotation's "Include/Exclude Mitigations" radio (ST + AoE) and the Advanced Mitigation feature, so it works whichever one you run. Changes apply immediately and persist.

This only flips the on/off switch for the whole category - the individual Advanced Mitigation skill options still need to be set up in WrathCombo first.
