# GatherBuddy Reborn tweaks

Quality-of-life tweaks for GatherBuddy Reborn's Vulcan crafting browser.

In the Recipes list you can right-click a recipe and pick **Create New List**, which opens a small menu with a "List name..." field and an "Ephemeral" checkbox. The field normally starts empty.

## Options

- **Prefill the new list name with the recipe name** (on by default) - fills the "List name..." field with the recipe's name the moment the menu opens, so most of the time you can just hit **Create & Add**. It only fills an empty field, so if you start typing your own name it stays out of the way.
- **Also tick "Ephemeral"** (off by default) - ticks the menu's "Ephemeral" checkbox for you each time it opens (the list is deleted automatically after crafting). Set only once per opening, so you're free to untick it.
- **New lists: tick "Skip if Already Have Enough"** (off by default) - any crafting list created while this is on gets the list editor's "Skip if Already Have Enough" option ticked automatically. Existing lists are left untouched, and each new list is set only once, so you can untick it afterwards.
- **Quest items overlay** (off by default) - shows a small, borderless panel anchored just left of the recipe search box, listing the craftable items your accepted quests require (e.g. the Black Pearl Ring for "Jaded"). Click an item to drop it straight into the search box. It only appears while that search box is on screen (Vulcan window open, not collapsed, Recipes tab), and only lists craftable items - quest items obtained through combat, gathering, etc. are skipped since they have no recipe.

GatherBuddy Reborn must be installed and loaded; reload YABOT after installing it.
